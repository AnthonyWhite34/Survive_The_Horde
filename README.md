Survive The Horde is my current Unreal Engine 5 project and learning sandbox. It began while following the Udemy course **"Unreal Engine 5 - Gameplay Ability System - Top Down RPG"**, and I am continuing to expand it with my own gameplay systems, experiments, and design ideas as I build deeper practical experience with Unreal Engine, C++, Blueprint workflows, and the Gameplay Ability System.

This project is my main portfolio project for now while I continue learning, improving the codebase, and working toward publishing a playable build.

## Project Goals

- Build a top-down action RPG / horde-survival experience in Unreal Engine 5.
- Practice and understand the Gameplay Ability System through real gameplay features.
- Add original systems and content beyond the course material to test my understanding.
- Improve my Unreal C++ architecture, Blueprint integration, UI, enemy behavior, and combat design skills.
- Grow the project into a stronger portfolio piece over time.

## Current Focus

The project currently includes and explores:

- Gameplay Ability System foundations, including attributes, ability system components, player state integration, and gameplay tags.
- Player and enemy character classes for RPG-style combat experimentation.
- UI widget controller and HUD work for displaying player information.
- Effect actors and pickups for attribute changes such as health and mana.
- Additional Unreal template/variant systems for combat, platforming, and side-scrolling experimentation.
- AI and StateTree-based systems used for enemy and gameplay behavior tests.

## Requirements

- Unreal Engine 5.7 or compatible Unreal Engine 5 version.
- Gameplay Ability System plugin enabled.
- StateTree and Gameplay StateTree plugins enabled.
- Enhanced Input and UMG support through the project module dependencies.

## Opening the Project

1. Clone or download the repository.
2. Open `Survive_The_Horde.uproject` with Unreal Engine.
3. If prompted, allow Unreal Engine to rebuild project files/modules.
4. Open the editor and compile the C++ project if needed.
